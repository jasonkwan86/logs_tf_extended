# logs_tf_extended
 [Firefox](https://addons.mozilla.org/en-CA/firefox/addon/logs_tf_extended/)/[Chrome](https://chromewebstore.google.com/detail/logstfextended/jcgpbbalnjcjedofegcdmeidnllgmapn) addon that adds ETF2L and RGL name and player division info to played games on [logs.tf](https://logs.tf/)


## Notes
The `"browser_specific_settings"` property must be deleted in the `manifest.json` file to test the unpacked extension on firefox
